# ✨ PlanejaFlow PRO

## App de Gestão Financeira com IA

### Funcionalidades
- 📊 **Dashboard** - Visão completa das finanças
- 💰 **Transações** - Controle de receitas e despesas
- 📈 **Investimentos** - Acompanhamento de investimentos
- 🎯 **Metas** - Defina metas com FOTOS da galeria
- 💳 **Cartões** - Gerencie cartões de crédito
- 🤖 **IA Inteligente** - Importe PDF e a IA sugere categorias
- 📄 **Relatórios PDF** - Exporte relatórios completos
- 🌙 **Modo Escuro** - Interface adaptativa
